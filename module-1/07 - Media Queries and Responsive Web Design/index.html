Media Queries & Responsive Web Design

What makes a website responsive?

Responsive web design requires three things:

  1. It must have a fluid layout.
  2. It needs flexible images.
  3. It should use media queries.

We've already introduced fluid layouts and flexible images, so let's dive right into media queries.

What are media queries?

Media queries allow us to apply different styles or use different layouts based upon whether or not a condition is met. This condition might be whether or not the screen is a certain size, if the user is printing the page, or if the user is holding their device in portrait or landscape orientation.

The condition in a media query always resolves as true (i.e. yes, this thing is happening) or false (i.e. no, this thing is not happening).

Most of the time, we're going to use media queries to check if the viewport is a certain size. If it is, we can change the layout and add additional rules; if not, all of the rules inside the media query will be ignored.

Taking a mobile-first design approach.

When building responsive websites, we need to take a mobile-first design approach. This means we must begin with creating an optimized experience for mobile devices before scaling up to larger screens such as tablets and desktop monitors.

There are a few reasons why taking a mobile-first is extremely important.

  1. It's much easier to add a few additional rules at the bottom of your stylesheet in order to create wider layouts rather than try to undo what you've already done to create narrower layouts.
  2. Mobile-first design often leads to better performance because it prioritizes efficiency and speed on mobile devices, which typically have more limited resources than desktops.
  3. As the vast majority of users have a mobile device, a mobile-first approach ensures that your website remains relevant and accessible to the majority of users.
  4. Search engines like Google prioritize mobile-friendly websites in their rankings, so a mobile-first design can improve your site's visibility and search engine rankings.
